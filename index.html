Wiki-Tube

A YouTube Wikipedia Mash-Up

This project was a midterm for my web programming class this past semester. Co-created with a classmate, Juan Navarro, this project displays the top five search results from youtube along with displaying the top wikipedia page, or the top ten links on wikipedia if the search was not specific enough. This incorporates the youtube and wikipedia data APIs. It also uses iFrame top allow the user to view youtube videos on the site. The thought behind this project was to give one simple, user friendly interface that combined basic encyclopedia information with pop culture references. We thought that the website would give a larger persepctive on topics, consolidating videos with encyclopedic information.

  • Date: October 2014
  • Contributions: Juan Navaro
  • APIs: YouTube, Wikipedia

Secret Santa

Generate secret santa for your friends or family!

This project was inspired by my group of friends. We all went to differenct schools across the country, but we thought it would be fun to do a secret santa with everyone. But, as we were so far away from each other it was impossible to draw names. So, I wrote a basic python program that generated random name pairings and then put them into separate text files. I then emailed them out to the respective "santas".

While rewriting the program to put it on the web, there were some problems uploading it to heroku, so I rewrote the program using HTML and Javascript, folowing a Model, View, Controller structure. The program allows the user to input as many names as necessary then allows them to download the text file for each person which includes their secret santa assignment. After this, they can email the appropriate text file to each participant.

Future improvements to be included are stylistic improvements and, theoretically, a login feature allowing users to upload a wish list and email so once they enter a group, they do not have to download text files but rather they recieve an email with their assignment and thier recipients wish list.

  • Created: December 2014
  • Languages used: HTML, CSS, Javascript
  • Repository: Secret Santa

Password Generator

Generate an easy to remember, hard to crack password

This password generator asks for parameters you wish to apply to your password then generates 10, easy to remember, 4-word passwords using common english words.This app is deployed using python and flask through Heroku.

Still some bugs being worked out. The app has difficulty running when the overall length is very close to 4x the maximum word length or the min and max word length are close to each other.

  • Created: November 2014
  • Platform: Heroku
